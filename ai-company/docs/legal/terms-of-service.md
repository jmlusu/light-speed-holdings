## 2. Description of Service

The AI Company Builder is a Python-based command-line tool and runtime framework for creating and orchestrating hierarchies of AI agents. The Service includes:

- **Agent Registry and Generator**: Tools for defining agent hierarchies in YAML and generating OpenCode-compatible agent configurations
- **Executor**: A multi-turn agentic loop implementing the ReAct pattern for LLM and tool interaction
- **Orchestrator**: Task management via MessageBus, scheduled operations via Scheduler, escalation via EscalationManager, and human approval via ApprovalGate
- **Cost Tracker**: LLM token usage tracking and budget enforcement
- **Model Router**: Multi-provider LLM routing with tiered fallback chains
- **CEO Dashboard**: A FastAPI-based web interface for monitoring company KPIs and operations
- **CLI Commands**: Command-line interface for managing agents, tasks, workflows, and system health

The Service is provided as open source software under the proprietary license described in Section 4. At version 1.0, the Service will transition to the Apache License 2.0.
